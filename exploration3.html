Welcome to my Exploration #3 where I am using VueJS. The idea behind this project was to so serve as a personal page for me/the user.

In my Exploration #1 assignment, I explored using tabs and displaying different content when switching between each tab. For this project, however, I wanted to go a bit further by making it a sort of personal "about me" page for me / someone that would be using it. It could also be used to serve as an online portfolio/resume page, and if this were the case, then these tabs could use used to store more relevant information like links to a LinkedIn page, a PDF of an actual resume and some information about the creator.
